## Running the Project Locally

```bash
# clone the repository
git clone https://github.com/ItzJohn2o1/Portfolio.git

cd Portfolio

# install dependencies
npm install

# start development server
npm run dev
```

The site will be available at the local Vite dev server URL shown in your terminal.

---

## Deployment

This project is built with Vite and deployed to GitHub Pages.

Basic steps:

1. Set the correct base path in `vite.config.js`:
   ```js
   base: '/Portfolio/'
   ```
2. Build and deploy:
   ```bash
   npm run build
   npm run deploy
   ```

GitHub Pages serves the site from the `gh-pages` branch at:  
`https://itzjohn2o1.github.io/Portfolio/`
